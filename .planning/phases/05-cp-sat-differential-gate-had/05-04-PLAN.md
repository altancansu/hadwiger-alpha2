<objective>
Build the differential agreement gate (`solvers/differential.py`) — the pure,
stdlib-only cross-examination that makes CBC/CP-SAT disagreement release-blocking
and is the SOLE component permitted to license an SHC-CANDIDATE. It consumes two
`ExactOutcome`s for the same instance and returns a verdict; unequal proven optima
raise `CriticalDisagreement` (quarantine + halt).

Purpose: EXACT-04. Phase 5 is the first phase where an impossibility-flavored
(had_2 < chi) claim becomes representable at all, and this gate is what makes such
a claim require TWO independent proofs. The differential is also guard 3 of 3 for
had_3 (a lost size-3 constraint shows as one backend > the other).
Output: `differential.py` + fast unit/integration tests + a solver-path -O canary.

This plan owns the ONE shared solver-path -O canary file (`test_solver_paths_dash_O.py`),
extending the -O discipline over the new CP-SAT and differential paths.
</objective>
