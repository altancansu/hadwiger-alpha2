|----------|-------------|
| CLI args -> pipeline | untrusted --family/--n/--seed cross into the runbook |

| T-06-05 | Tampering (false RESISTANT) | pipeline heuristic-miss handling | mitigate | miss (sets is None) is an unconditional edge to exact; RESISTANT reachable ONLY via exact INSUFFICIENT/timeout |
