<!-- The subprocess idiom to copy EXACTLY (RESEARCH Pattern 1 / 07-PATTERNS.md §generate.py): -->
- `stream_mtf(n, res=None, mod=None) -> Iterator[(index, graph6, shard)]`
  p1 = subprocess.Popen(["geng", "-ctq", str(n)] + ([shard] if shard else []), stdout=PIPE)
  p2 = subprocess.Popen(["pickg", "-Z2"], stdin=p1.stdout, stdout=PIPE, text=True)
  p1.stdout.close(); iterate p2.stdout lines; p2.wait()
  shard string is "res/mod" (e.g. "0/64") only when mod is set.
- `geng_version()` — capture nauty version from `geng --help` stderr/stdout first line, for provenance.
