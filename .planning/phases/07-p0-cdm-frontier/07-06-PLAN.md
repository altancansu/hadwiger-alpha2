## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| graph6 line → decoded graph | untrusted-but-bounded external input; decoded n must equal requested n |
| dual solver verdicts → adjudication | DFS (arbiter) vs CP-SAT (cross-check); disagreement is radioactive and release-blocking |
| CDM-FAIL → escalation | connected-complement fail is a potential Hadwiger event (radioactive); disconnected fail is benign/out-of-scope |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-7-diff | Tampering (soundness) | DFS≡CP-SAT disagreement silently resolved | mitigate | CriticalDisagreement raised on any mismatch → quarantine + HALT batch; never "best of two" (differential.py stance) |
| T-7-13 | Spoofing/Tampering | disconnected-complement CDM-fail mis-escalated as a Hadwiger event | mitigate | `nx.is_connected(complement)` classification BEFORE escalation; K_a⊔K_b catalogued out-of-scope, no battery/E3 call (Pitfall 1 / Q1) |
| T-7-10 | Tampering | graph6 with n ≠ requested decoded and adjudicated | mitigate | assert `H.number_of_nodes() == n` at the decode boundary before building adj |
| T-7-14 | Tampering (integrity) | miscounting the frontier (dropped shard / bad res-mod) | mitigate | assert batch total == 1,813; sharding-sum guard from 07-04; corpus chain re-verifies end to end |
